Face Dermatology Disease Detection using Machine Learning and Image processing

  • Industry: Healthcare
  • Domains: Machine Learning, Deep Learning, Image Processing, Computer Vision
  • Duration: March 2021- June 2021
  • Authors: Pratik Patil, Prajyot Tayade, Atharva Shirgave, Prathamesh Pimparwar, Kaustubh Wagh
  • Institute: Pimpri Chinchwad College of Engineering, Nigdi, Pune

Description- Face Dermatology Diseases are increasing today because of environmental problems. The current disease detection system fails to detect disease with greater accuracy because they do not use optimal solutions and is only available only at hospitals. Sometimes doctors acquire more charges in the name of fake disease. So, we have proposed a system which will detect disease with more accuracy and will be user friendly so it can be used by anyone. We have proposed a greedy algorithm for disease detection which will select optimal solutions at each step.

Modern Web Engineering

  • Industry: Project Management
  • Domains: Web Development
  • Duration: March 2019- September 2019
  • Authors: Pratik Patil, Mohini Shinde
  • Institute: Government Polytechnic, Karad

Description- Web Engineering is a quantifiable approach for development of Web Applications. We studied the current Web Engineering process. Web Engineering involves set of activities that are carried out to accomplish Web Development project. After studying the Web Engineering process we found some issues regarding the process that is followed currently. Based on those we suggested some solutions to solve those issues. For analyzing the results by implementing the proposed Web Development strategy, we developed some projects by following traditional approach and developed the same by following the proposed approach which we named as Modern Web Engineering. Then we compared the results, Modern Web Engineering solved all the problems in the existing system.
